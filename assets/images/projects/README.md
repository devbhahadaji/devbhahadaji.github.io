## Image Guidelines

### Web Project Screenshots (Landscape)
- **Recommended size**: 1200x800px or 16:10 aspect ratio
- **Format**: PNG or JPG (optimized for web)
- **Featured image**: Should be the most impressive view of your project

### Mobile App Screenshots (Portrait)
- **Recommended size**: 390x844px (iPhone 14 size) or similar portrait ratio
- **Format**: PNG with transparency if needed
- **Add `type: "portrait"` in the gallery YAML configuration**

### Tips
1. Use high-quality screenshots
2. Compress images before adding (use tools like TinyPNG)
3. Ensure consistent styling across gallery images
4. For app screenshots, consider using device mockups for better presentation
